Для запуска программы необходимо запустить файл main.py

Я использовал базу данных postgresql с именем dnstest, имя базы находится в файле
database.py, в переменной db_name. Для работы программы необходимо создать базу
данных с именем dnstest (либо своим собственным), указать действительные
host и port сервера postgresql, а так же поместить файлы с исходными данными в папку src
в корне проекта.
